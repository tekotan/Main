Sentiment Analysis

Duration: June 20, 2018 - July 10, 2018
Partner: None Platform: IPython
Objective: To test the best neural network type for text based classifications
Details: For this project, I wanted to determine the best network architecture for text based classifications. I used the most popular Deep Learning techniques, such as Recurrent Neural Networks, Convolutional Neural Networks, and Densely Connected Layers. Because of my technology limitation of having a budget laptop, I chose the more simpler task of classifying the sentiment of the text as opposed to something like text autocompletion or a translator. The other reason I chose classification becasue there was a large amount of data available for the task, this data coming from twitter. The main challenge in this project was learning how to process all of the data in a way that could be easily read in python. I worked around this by using the internet to research ways to massage the data. The main skills I learned during this process was how to stay focused on a task despite hiccups, and real time experience on the process of building an algorithm using tensorflow.
